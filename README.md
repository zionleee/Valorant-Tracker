# Valorant Performace Tracker

## Application Features & Description
This application will be a **player performance tracking application** for Valorant players.
Players will be able to record their competitive matches played for a certain date. 
Each match will be recorded with the following information: *Win/Loss, K/D/A, ACS score, agent selected and map.* 
The recorded matches will then be saved onto their performance profile.
Players will be able to view their Valorant **Match History**, **statistics on win rate**, and **statistics on K/D**.

## User Stories
- User wants to record Valorant Matches played 
- User wants to view Match History
- User wants to search for all Matches played on specific date
- User wants to view statistics on their win ratio
- User wants to view Statistics on their K/D
